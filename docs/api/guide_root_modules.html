Processing process.py — 点云处理与可视化渲染核心

负责将 Fast3R 模型输出转化为 VTK 可渲染的点云，包含坐标对齐、天空遮罩、颜色映射等处理步骤。

主要函数

start_visualization(window, output_dict, min_conf_thr_percentile, global_conf_thr_value_to_drop_view, point_size)
可视化主入口，由 logic_mainwindow.py 调用。调用 frame_processing 再调用 reconstruction。
align_local_pts3d_to_global(preds, views, min_conf_thr_percentile)
将每个视图的局部点云（pts3d_local）通过刚体变换（含缩放）对齐到全局坐标系。使用 roma.rigid_points_registration 求解 R、t、s，并行处理所有视图。结果写入 preds[i]['pts3d_local_aligned_to_global']。
frame_processing(output, gui_global_conf_threshold)
逐帧提取点云、颜色（原始RGB / 置信度热图 / 彩虹色）、天空遮罩，并计算场景空间范围 max_extent。
reconstruction(window, frame_data_list, max_extent, ...)
将处理后的帧数据转换为 VTK Actor 并添加到渲染器。根据 is_outdoor 决定是否启用天空遮罩。
detect_sky_mask(img_rgb)
基于 HSV 颜色空间（蓝天/白云）检测天空像素，使用形态学操作进行区域清理。返回非天空区域的二值掩码。
is_outdoor_scene(frame_data_list)
统计超过 20% 天空像素的帧比例，若超过 25% 则判定为室外场景。
on_left_click(obj, event, window, points, max_extent)
VTK 鼠标点击回调。找到最近点后绘制红色球体标记；若处于两点模式（radioButton_4），则额外绘制两点之间的连线。

可调参数（在 logic_mainwindow.py 中传入）

参数默认值说明
min_conf_thr_percentile10对齐时使用的最低置信度百分位（越低保留点越多）
global_conf_thr_value_to_drop_view1.5全局置信度低于此值的视图将被丢弃
point_size0.0004点云渲染点的大小
UI Helper images_upload_display.py — 图片缩略图展示

在 UI 的 widget_2 区域以滚动网格形式展示已上传的图片缩略图（每行 6 张）。

❓ 常见问题

Q: 启动时报 ModuleNotFoundError

确保已安装所有依赖并激活正确的 conda 环境，参考项目 requirements.txt：

conda activate omni3d
pip install -r requirements.txt

Q: 重建后点云没有显示

检查 global_conf_thr_value_to_drop_view 参数，若设置过高会丢弃所有视图。默认值 1.5 适合大多数场景，可尝试降低到 1.0。

Q: 视频上传后没有图片预览

视频帧会提取到 temp_preview_frames/preview/，若该目录权限问题可手动创建。视频预览播放依赖 QMediaPlayer，需要系统安装对应解码器。

Q: 不要手动编辑 MainWindow.py

该文件由 pyuic5 自动生成，手动修改会在下次重新生成时被覆盖。界面修改请通过 Qt Designer 编辑 MainWindow.ui。
